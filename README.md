# Agencia-Viajes
Es una página web para proyecto de Agencia de viajes, de la semana 4 del módulo programación WEB II.
Requerimiento:
Tarea semana , módulo Programación WEB II.
1.Implementa una función en PHP que genere notificaciones emergentes para informar a los usuarios sobre ofertas especiales al acceder a la página web. Escribe el fragmento de código PHP y los ajustes respectivos realizados al HTML.
2. Escribe el fragmento de código PHP correspondiente a la creación de una clase que represente un filtro interactivo para destinos y fechas de viaje, con sus propiedades (nombre del hotel, ciudad, país, fecha de viaje, duración del viaje) y métodos que faciliten su gestión y permitan a los usuarios realizar búsquedas personalizadas.
3. Escribe el fragmento de código HTML para implementar un formulario en PHP a partir de las propiedades de la clase creada en la actividad anterior para el registro de destinos y fechas correspondientes a la intención de viaje del turista.
4. Escribe el fragmento de código HTML y el script en PHP aplicando un método de recuperación de datos y paso de variables desde el formulario de intención de viaje diseñado.
